FileCairn Tutorial

Step-by-step guide to run, search, and edit the FileCairn project page.

1. Build and Run Locally

Open Terminal and run:

cd /Users/kevin/filecairn
go build -o ./bin/filecairn ./cmd/filecairn
./bin/filecairn init

2. Scan and Search

After setup, scan files and query by keywords:

./bin/filecairn scan --all
./bin/filecairn find "toefl test"
./bin/filecairn status

Tip: use specific query phrases like toefl reading test 1 for cleaner results.

3. Use API Mode

Start API service for tool integration:

./bin/filecairn serve --addr 127.0.0.1:4819
curl -sS http://127.0.0.1:4819/v1/health
curl -sS http://127.0.0.1:4819/v1/find -H 'Content-Type: application/json' -d '{"query":"filecairn","limit":5}'

4. Install Skills

After API mode is running, install skill files so tools can call FileCairn directly:

Install skill files:

mkdir -p "$HOME/.filecairn/skills"
cp /Users/kevin/filecairn/skills/*.md "$HOME/.filecairn/skills/"

5. Register Created Files

When a tool writes a file, register it immediately:

echo "new note" > /tmp/new-note.md
./bin/filecairn register-created --path /tmp/new-note.md --agent openclaw --share full
./bin/filecairn find "new note"

FileAtlas Tutorial

Step-by-step guide to run, search, and edit the FileAtlas project page.

1. Build and Run Locally

Open Terminal and run:

cd /Users/kevin/fileatlas
go build -o ./bin/fileatlas ./cmd/fileatlas
./bin/fileatlas init

2. Scan and Search

After setup, scan files and query by keywords:

./bin/fileatlas scan --all
./bin/fileatlas find "toefl test"
./bin/fileatlas status

Tip: use specific query phrases like toefl reading test 1 for cleaner results.

3. Use API Mode

Start API service for tool integration:

./bin/fileatlas serve --addr 127.0.0.1:4819
curl -sS http://127.0.0.1:4819/v1/health
curl -sS http://127.0.0.1:4819/v1/find -H 'Content-Type: application/json' -d '{"query":"fileatlas","limit":5}'

4. Register Agent-Created Files

When a tool writes a file, register it immediately:

echo "new note" > /tmp/new-note.md
./bin/fileatlas register-created --path /tmp/new-note.md --agent openclaw --share full
./bin/fileatlas find "new note"

5. Edit Your Website Project Page

  1. Edit this file: /Users/kevin/Documents/GitHub/liyuanda/public/project-fileatlas.html
  2. Refresh browser page: /liyuanda/project-fileatlas.html
  3. If updates do not appear, do hard refresh: Cmd + Shift + R

6. Save to Git

Save your website changes:

cd /Users/kevin/Documents/GitHub/liyuanda
git add public/project-fileatlas.html public/fileatlas-tutorial.html public/data/projects.js
git commit -m "Add FileAtlas page and tutorial"
git push origin main
